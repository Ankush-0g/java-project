# My Java Projects Collection

Welcome to my Java development repository! This repository serves as a centralized portfolio containing various Java applications, ranging from core programming logic to advanced data structures and mini-projects.


---

## 🛠️ Prerequisites & Tech Stack

To run any of the projects in this repository, ensure you have the following installed:

*   **Java Development Kit (JDK):** Version 17 or higher
*   **Integrated Development Environment (IDE):** [IntelliJ IDEA](https://jetbrains.com), [Eclipse](https://eclipse.org), or [VS Code](https://visualstudio.com)
*   **Build Tool (Optional):** Maven or Gradle (if applicable to specific sub-folders)

---

## 🚀 How to Run the Projects Locally

Follow these steps to clone the repository and run a project on your local machine:

### 1. Clone the Repository
```bash
git clone https://github.com
```

### 2. Navigate to the Main Directory
```bash
cd java project
```

### 3. Compile and Run a Specific Project
Navigate to the specific project folder containing your `Main.java` file:
```bash
cd ...
javac Main.java
java Main
```

## 🧠 Core Concepts Covered

These projects highlight key Java programming milestones and fundamentals:
*   **Object-Oriented Programming (OOP):** Encapsulation, Inheritance, Polymorphism, and Abstraction.
*   **Exception Handling:** Managing runtime errors gracefully using `try-catch` blocks.
*   **Java Collections Framework:** Working with `List`, `Set`, `Map`, `ArrayList`, and `HashMap`.
*   **File I/O:** Reading from and writing to external text/data files.
